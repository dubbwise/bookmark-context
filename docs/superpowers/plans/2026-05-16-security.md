# Security Fixes Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Eliminate SSRF via unvalidated URLs, add a static auth token so only the extension can call the daemon, add a request body size cap, and fix the `escHtml` attribute-injection gap in the side panel.

**Architecture:** A new `bookmark_context/auth.py` module generates and persists a random token on first startup. The daemon loads it into `Config` and enforces it via a FastAPI middleware (skipping only `/status`). The extension reads the token from `chrome.storage.sync` and sends it as `Authorization: Bearer <token>` on every request. URL validation is a Pydantic `field_validator` on `BookmarkCreate`; body size is a middleware. The extension `escHtml` gets `"` and `'` added.

**Tech Stack:** Python `secrets` stdlib, Pydantic v2 `field_validator`, FastAPI middleware, Chrome `chrome.storage.sync`

## Known limitation: pre-save content scanning (May 2026)

> **Status:** Scanning likely does **not** work as users expect on many real pages. Treat the pre-save scan gate as best-effort, not a security guarantee.

The injection / LLM-targeted-instruction scanner (`bookmark_context/indexer/scanner.py`) only runs on **plain text extracted** from a bookmark at add time (`bookmark_context/api/bookmarks.py` → `scrape_url` → `chunk_text` → `scan_chunks`). That pipeline has important gaps:

| Issue | Effect |
|-------|--------|
| **JavaScript SPAs** (e.g. Ashby job boards) | Server-side HTTP fetch often yields only “You need to enable JavaScript…”, not the rendered form. Without browser-captured HTML, the honeypot text is never scanned. |
| **Failed HTML capture** | If `executeScript` / content-script capture fails, `html` is null and the daemon falls back to server fetch (same SPA problem). |
| **Empty extract skips scan** | When `scrape_url` returns `""`, the gate does not run at all and the bookmark saves with no warning (`if text:` in `bookmarks.py`). |
| **Regex-only detection** | Scanner uses pattern lists, not an LLM. Novel or obfuscated “instructions for AI” wording may score 0 until patterns are extended (see `llm_targeted_instruction` category). |
| **Post-save indexing** | `IndexPipeline` records `injection_risk` per chunk after save; that metadata does not block the initial add if the pre-save gate never saw the risky text. |

**Mitigations added after this plan was written** (not part of the tasks below): `capturePageHtml` fallbacks in the extension (`executeScript` → content script `GET_HTML` → `innerText`), SPA-shell detection (`content_unscannable` scan warning), and plain-text fallback when trafilatura fails on browser HTML. **Restart the daemon and reload the extension** after changing scanner/scraper code.

**Follow-up work** (if scanning should be reliable): pass pre-extracted text from add gate into `IndexPipeline` (see architecture plan Task 5), scan on empty extract instead of skipping, integration tests against real SPA URLs with captured HTML fixtures, and document that “force save” bypasses all checks.

from __future__ import annotations
import secrets
from pathlib import Path

_DEFAULT_TOKEN_PATH = Path.home() / ".config" / "bookmark-context" / "token"


def load_or_create_token(path: Path = _DEFAULT_TOKEN_PATH) -> str:
    path.parent.mkdir(parents=True, exist_ok=True)
    if path.exists():
        return path.read_text().strip()
    token = secrets.token_urlsafe(32)
    path.write_text(token)
    return token
```

from __future__ import annotations
from fastapi import FastAPI, Request
from fastapi.middleware.cors import CORSMiddleware
from starlette.responses import JSONResponse
from bookmark_context.config import Config
from bookmark_context.storage.database import Database
from bookmark_context.storage.vector_store import VectorStore
from bookmark_context.indexer.embedder import Embedder

VERSION = "0.1.0"
_MAX_BODY_BYTES = 10 * 1024 * 1024  # 10 MB


def create_app(config: Config) -> FastAPI:
    db = Database(config.db_path)
    db.init()
    vs = VectorStore(config.chroma_path)
    embedder = Embedder(config.embed_model)

    app = FastAPI(title="bookmark-context")
    app.state.db = db
    app.state.vs = vs
    app.state.embedder = embedder
    app.state.config = config

    app.add_middleware(
        CORSMiddleware,
        allow_origins=["*"],
        allow_methods=["*"],
        allow_headers=["*"],
    )

    @app.middleware("http")
    async def limit_body_size(request: Request, call_next):
        content_length = request.headers.get("content-length")
        if content_length and int(content_length) > _MAX_BODY_BYTES:
            return JSONResponse({"detail": "Request body too large"}, status_code=413)
        return await call_next(request)

    @app.middleware("http")
    async def require_token(request: Request, call_next):
        if not config.api_token:
            return await call_next(request)
        if request.url.path == "/status":
            return await call_next(request)
        auth = request.headers.get("Authorization", "")
        if not auth.startswith("Bearer ") or auth[7:] != config.api_token:
            return JSONResponse({"detail": "Unauthorized"}, status_code=401)
        return await call_next(request)

    @app.get("/status")
    def status():
        return {"status": "ok", "version": VERSION}

    from bookmark_context.api.collections import router as collections_router
    from bookmark_context.api.bookmarks import router as bookmarks_router
    app.include_router(collections_router)
    app.include_router(bookmarks_router)

    return app
```

from __future__ import annotations
from urllib.parse import urlparse
from pydantic import BaseModel, field_validator


class CollectionCreate(BaseModel):
    name: str
    description: str = ""


class CollectionResponse(BaseModel):
    id: str
    name: str
    description: str
    created_at: str
    updated_at: str
    bookmark_count: int = 0


class BookmarkCreate(BaseModel):
    url: str
    title: str = ""
    html: str | None = None

    @field_validator("url")
    @classmethod
    def url_must_be_http(cls, v: str) -> str:
        parsed = urlparse(v)
        if parsed.scheme not in ("http", "https"):
            raise ValueError("URL must use http or https scheme")
        if not parsed.netloc:
            raise ValueError("URL must have a valid host")
        return v


class BookmarkResponse(BaseModel):
    id: str
    collection_id: str
    url: str
    title: str
    favicon_url: str = ""
    added_at: str
    indexed_at: str | None
    index_status: str
    error_message: str | None = None

```js
async function getDaemonBase() {
  const { daemonPort = 7331 } = await chrome.storage.sync.get("daemonPort");
  return `http://localhost:${daemonPort}`;
}

async function request(method, path, body = null) {
  const base = await getDaemonBase();
  const { apiToken = "" } = await chrome.storage.sync.get("apiToken");
  const opts = {
    method,
    headers: {
      "Content-Type": "application/json",
      ...(apiToken ? { "Authorization": `Bearer ${apiToken}` } : {}),
    },
  };
  if (body !== null) opts.body = JSON.stringify(body);
  const res = await fetch(`${base}${path}`, opts);
  if (!res.ok) {
    const text = await res.text().catch(() => "");
    throw new Error(`${method} ${path} → ${res.status}: ${text}`);
  }
  if (res.status === 204) return null;
  return res.json();
}

export const api = {
  status: () => request("GET", "/status"),
  listCollections: () => request("GET", "/collections"),
  createCollection: (name, description = "") =>
    request("POST", "/collections", { name, description }),
  deleteCollection: (id) => request("DELETE", `/collections/${id}`),
  listBookmarks: (collectionId) =>
    request("GET", `/collections/${collectionId}/bookmarks`),
  addBookmark: (collectionId, url, title, html = null) =>
    request("POST", `/collections/${collectionId}/bookmarks`, { url, title, html }),
  deleteBookmark: (id) => request("DELETE", `/bookmarks/${id}`),
  reindexBookmark: (id) => request("POST", `/bookmarks/${id}/reindex`),
};
```
