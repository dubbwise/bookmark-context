# Bug Fixes Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix four confirmed bugs: the "Online · undefined" status text in the side panel, the `favicon_url` field that is never populated, a race condition where two concurrent index jobs can run for the same bookmark, and the context menu path silently discarding scan warnings (shows green ✓ badge when page was NOT saved).

**Architecture:** The status text fix is a one-line JS change. Favicon extraction is added as a new function in `scraper.py` and wired into `pipeline.py`; the DB gains a `favicon_url` column update in `update_bookmark_status`. The reindex race is fixed by checking `index_status` before queuing a new job. The context menu scan warning is fixed by inspecting the API response in `service_worker.js` and opening the side panel to let the user decide.

**Tech Stack:** Python stdlib `re`/`urllib.parse`, existing pytest/httpx/respx test stack, vanilla JS

```python
import re
from urllib.parse import urljoin, urlparse


def extract_favicon(html: str, url: str) -> str:
    """Return favicon URL from HTML <link rel="icon">, falling back to /favicon.ico."""
    patterns = [
        r'<link[^>]+rel=["\'](?:shortcut )?icon["\'][^>]*href=["\']([^"\']+)["\']',
        r'<link[^>]+href=["\']([^"\']+)["\'][^>]*rel=["\'](?:shortcut )?icon["\']',
    ]
    for pattern in patterns:
        match = re.search(pattern, html, re.IGNORECASE)
        if match:
            return urljoin(url, match.group(1))
    parsed = urlparse(url)
    return f"{parsed.scheme}://{parsed.netloc}/favicon.ico"
```

Note: `re` and `urllib.parse` are already part of the Python stdlib — no new dependencies.

```python
from __future__ import annotations
from bookmark_context.storage.database import Database
from bookmark_context.storage.vector_store import VectorStore
from bookmark_context.indexer.embedder import Embedder
from bookmark_context.indexer.scraper import scrape_url, extract_favicon
from bookmark_context.indexer.chunker import chunk_text


class IndexPipeline:
    def __init__(self, db: Database, vs: VectorStore, embedder: Embedder) -> None:
        self.db = db
        self.vs = vs
        self.embedder = embedder

    def index_bookmark(self, bookmark_id: str, html: str | None = None) -> None:
        bm = self.db.get_bookmark(bookmark_id)
        if not bm:
            return

        self.db.update_bookmark_status(bookmark_id, "indexing")
        try:
            if html:
                text = scrape_url(bm["url"], html=html)
                favicon_url = extract_favicon(html, bm["url"])
            else:
                import httpx
                raw_html = None
                try:
                    resp = httpx.get(
                        bm["url"], timeout=15, follow_redirects=True,
                        headers={"User-Agent": "Mozilla/5.0 (compatible; BookmarkContext/0.1)"}
                    )
                    if resp.status_code < 400:
                        raw_html = resp.text
                except (httpx.RequestError, httpx.HTTPStatusError):
                    pass
                text = scrape_url(bm["url"], html=raw_html) if raw_html else ""
                favicon_url = extract_favicon(raw_html, bm["url"]) if raw_html else f"https://{bm['url'].split('/')[2]}/favicon.ico"

            if not text:
                self.db.update_bookmark_status(bookmark_id, "error", error_message="No content extracted from page")
                return

            chunks = chunk_text(text)
            embeddings = self.embedder.embed(chunks)

            chroma_ids = [f"{bookmark_id}-{i}" for i in range(len(chunks))]
            metadatas = [
                {"url": bm["url"], "title": bm["title"], "bookmark_id": bookmark_id}
                for _ in chunks
            ]
            self.vs.add_chunks(
                collection_id=bm["collection_id"],
                chroma_ids=chroma_ids,
                texts=chunks,
                embeddings=embeddings,
                metadatas=metadatas,
            )
            self.db.add_chunks(bookmark_id, list(zip(chunks, chroma_ids)))
            self.db.update_bookmark_status(bookmark_id, "done", favicon_url=favicon_url)
        except Exception as e:
            self.db.update_bookmark_status(bookmark_id, "error", error_message=str(e))
```

Note: the original `scrape_url` already fetched via httpx; this refactor fetches once and uses the HTML for both text extraction and favicon, avoiding a double HTTP request.

```python
def reindex_bookmark(bookmark_id: str, request: Request, background_tasks: BackgroundTasks):
    db = request.app.state.db
    bm = db.get_bookmark(bookmark_id)
    if not bm:
        raise HTTPException(status_code=404, detail="Bookmark not found")
    if bm["index_status"] == "indexing":

    return db.get_bookmark(bookmark_id)
```

### Task 5: Fix context menu scan warning (silent discard)

**Files:**
- Modify: `extension/background/service_worker.js`

No automated test — manual verification only.

**Context:** `service_worker.js` calls `api.addBookmark(...)` on right-click but never checks the response body. When the API returns `{ status: "scan_warning" }` (HTTP 200, nothing saved), the service worker assumes success and shows a green ✓ badge. The page is silently not saved and the user has no way to act on the warning.

The fix: check the response. If `status === "scan_warning"`, clear the tab badge, store the pending save data in `chrome.storage.session`, and open the side panel — which already has the scan warning dialog. The side panel reads the pending data on load and shows the dialog automatically.

- [ ] **Step 1: Store pending scan data and open side panel from `service_worker.js`**

In `extension/background/service_worker.js`, replace the `api.addBookmark` call and success handling inside `chrome.contextMenus.onClicked.addListener`:

```js
  try {
    const result = await api.addBookmark(collectionId, tab.url, tab.title || tab.url, html);

    if (result && result.status === "scan_warning") {
      // Store warning data for the side panel to pick up
      await chrome.storage.session.set({
        pendingScanWarning: {
          collectionId,
          url: tab.url,
          title: tab.title || tab.url,
          html,
          warning: result,
        },
      });
      await chrome.sidePanel.open({ tabId: tab.id });
      return;
    }

    // Normal success
    await chrome.action.setBadgeText({ text: "✓", tabId: tab.id });
    await chrome.action.setBadgeBackgroundColor({ color: "#22c55e", tabId: tab.id });
    setTimeout(() => {
      chrome.action.setBadgeText({ text: "", tabId: tab.id });
    }, 2000);
  } catch (e) {
    console.error("Failed to add bookmark:", e);
  }
```
